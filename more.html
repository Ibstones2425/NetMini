Install NetMini

Add NetMini to your device for a full-screen, app-like experience — no app store required.

Running as installed app

iOS — Safari

iPhone / iPad
  1. Open NetMini in Safari (other iOS browsers can't install PWAs).
  2. Tap the Share button at the bottom of Safari.
  3. Scroll down and select Add to Home Screen.
  4. Tap Add — NetMini launches as a standalone fullscreen app, hidden from the browser URL bar.

Android — Chrome

Pixel / Galaxy / etc.
  1. Open NetMini in Chrome (or Edge / Firefox for Android).
  2. Tap the browser options menu icon (⋮) or look for the Install App badge in the address bar.
  3. Select Install app / Add to Home screen.
  4. Confirm — NetMini downloads to your device drawer and launches in fullscreen.

Desktop — Chrome / Edge

Windows / macOS / Linux
  1. Open NetMini in Chrome or Microsoft Edge.
  2. Click the Install icon in the address bar (right side) — or open the menu → Install NetMini.
  3. Confirm the install dialog.
  4. NetMini opens in its own standalone window, no browser chrome.
